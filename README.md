## ✨ Main Features

### 🔁 Rule-based workflows

Set up simple rules that tell the agent what to do. This helps keep tasks in a fixed order and cuts down on guesswork.

### 🧠 Claude-powered agents

Use Claude as the engine for the AI agent. This gives the app the ability to handle text tasks, follow steps, and respond to your input.

### 🧩 Skill management

Add and organize skills for different jobs. You can keep skills grouped by task so the agent stays focused.

### 🧱 Clean JS and TS structure

The app uses a clear JavaScript and TypeScript layout. That makes the project easier to follow, extend, and run in a structured way.

### 📂 Modular design

Each part of the system works as a separate module. This helps keep the app tidy and makes it easier to change one part without breaking the rest.

### 🔍 Simple control flow

You can build agent logic in a way that is easy to read. This is useful when you want the same task to run the same way each time.
